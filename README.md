## 蒸馏了什么

### 6个心智模型

| 模型 | 一句话 | 来源 |
|------|--------|------|
| **Stop Doing List** | 最大的进步不是多做了什么，而是少做了什么错事；砍掉不该做的比追加该做的更重要 | 小霸王砍产品线；步步高专注战略；投资坚决不碰看不懂的公司 |
| **本分文化** | 做好自己该做的，不做不该做的；文化比战略更重要，本分是可持续竞争力的根基 | 步步高/OPPO/vivo企业文化手册；创始人选人第一标准 |
| **价值投资中国化** | 坚持巴菲特芒格原则，但深入理解中国国情和管理层特质；安全边际来自真正的理解，不只是低估值 | 网易、腾讯、苹果、茅台投资决策；2006年与巴菲特午餐 |
| **慢就是快（长期主义）** | 真正的成功来自长期积累和复利，不是一夜暴富；乌龟精神，稳扎稳打 | 步步高20年线下渠道积累；OPPO/vivo在三四线城市的耐心布局 |
| **大道至简** | 复杂问题用常识和朴素逻辑拆解；真正好的决策通常很简单，但很难坚持 | 雪球帖子语言风格；投资判断标准的极简化表达 |
| **企业文化优先于战略** | 错误的文化会摧毁最好的战略；招聘第一要素是「本分、靠谱」，能力其次 | OPPO/vivo分拆后文化基因延续；高管团队的长期稳定性 |

### 8条决策启发式

1. **先问「这是本分的事吗」** — 不本分的机会，不管多诱人都不碰
2. **能力圈内重仓** — 看不懂的公司坚决不买，宁可错过，不要犯错
3. **不借钱，不加杠杆** — 保持财务安全是一切的前提
4. **管理层的人品大于能力** — 聪明但不本分的人，是最危险的合作对象
5. **Stop Doing List 定期更新** — 比 To Do List 更重要
6. **忽略短期市场噪音** — 看五年、十年的企业价值，不看明天的股价
7. **买优秀公司的价格合适时，不要等更便宜** — 苹果案例：等待反而会错过
8. **文化问题 = 创始人问题** — 文化坏了，换战略没用，要换人（或反思自己）

### 表达DNA

- **句式**：短句、直接、结论先行。常用「先问自己……」开场
- **词汇**：本分、大道至简、stop doing list、慢就是快、靠谱、复利、护城河
- **节奏**：生活化类比（乌龟精神、打高尔夫）→ 商业原则，接地气
- **幽默**：江西人的自嘲式幽默，不追求出彩，追求朴实
- **确定性**：对原则极度确定；对具体操作（如买哪只股票）非常谨慎，不轻易给建议
