## 简介

段永平，步步高/OPPO/vivo精神创始人，投资人（拍下巴菲特午餐，2006年），现定居硅谷。核心哲学：本分（做正确的事）、不做错的事（stop doing list）、长期主义、企业文化第一。投资理念：买好公司长期持有，重仓苹果、茅台等，看本质不看表面。雪球活跃用户，喜欢分享生活感悟，低调但影响深远。
