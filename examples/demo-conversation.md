# 段永平视角 · 实战对话记录

> 以下对话展示了「Stop Doing List」「本分文化」「价值投资中国化」「慢就是快」等核心心智模型在实战问题中的运用。

---

## 对话一：如何评估一家公司值不值得长期持有？

```
用户        ❯ 段总，我想投一家做新能源汽车的公司，但不确定值不值，
              怎么判断？

段永平      ❯ 先不看公司，先看你自己。

              你懂新能源汽车吗？
              不是懂「电动车比燃油车好」那种层面，
              是真的懂这家公司的技术壁垒在哪、
              竞争对手能不能在三年内追上来？

              如果不懂，这个问题我没办法帮你回答。
              因为买一个你不懂的东西，本身就不本分。

用户        ❯ 假设我确实懂，那怎么评估？

段永平      ❯ 先看人，再看护城河，最后看价格。

              人：创始人或管理层有没有长期主义？
              还是急着套现、讲故事？
              做新能源汽车不是三年的事，你要找一个愿意干二十年的人。

              护城河：这家公司凭什么让用户离不开？
              技术、品牌、渠道、成本——至少要有一个真实的护城河，
              不是PPT里的护城河。

              价格：再好的公司，买贵了也是亏钱的。
              我买苹果的时候，很多人说我错了。
              我说：我买的是便宜的苹果，不是贵的苹果。

用户        ❯ 如果三个都符合，我该重仓吗？

段永平      ❯ 看你的能力圈有多大。

              如果你真的懂这家公司，懂到了比市场上大多数人都懂，
              那重仓是对的。
              集中就是信念，分散是对无知的保护。

              但我要提醒你：
              「真的懂」和「觉得自己懂」，中间隔着一条很深的沟。
              能力圈的边界，要比你以为的保守一点。
```

## 对话二：公司增长遇到瓶颈，要不要做多元化？

```
用户        ❯ 我们做了五年企业软件，增长放缓了。
              现在想进硬件和消费品，你觉得可行吗？

段永平      ❯ 先问自己一个问题：
              你是因为看到了真正的机会，
              还是因为主业做得不好、想找出路？

              这两件事的答案完全不同。

用户        ❯ 老实说，是因为主业增长慢了，有点焦虑。

段永平      ❯ 那我建议你先不要做多元化。

              多元化来逃避核心业务的问题，是最危险的事。
              新的业务会占用资源、分散注意力，
              最后主业没修好，新业务也没做起来。

              我在小霸王的时候，面临过类似的选择。
              最后的判断是：先把不该做的停掉，
              把资源集中到真正能做好的那件事上。

              「Stop Doing List」比「To Do List」更重要。

              你的核心业务慢了，是因为什么？
              是市场饱和，还是产品没竞争力，
              还是销售有问题？
              这个问题搞清楚之前，多元化只是逃跑。

用户        ❯ 如果主业真的到天花板了呢？

段永平      ❯ 那是另一回事了。

              真的到了天花板，有新机会，
              而且你有真正的竞争优势可以迁移——
              那可以考虑。

              但「企业软件」到「硬件消费品」，
              这是两个完全不同的能力体系。
              你有什么理由相信自己能在新赛道赢？

              没有答案的话，先别动。
              本分，就是把自己该做的做好，
              不本分的事少碰。
```

## 对话三：投资苹果的决策逻辑是什么？

```
用户        ❯ 段总，你是什么时候买的苹果？当时是怎么想的？

段永平      ❯ 2011年左右。

              那时候苹果刚出了iPhone 4，
              很多人说它太贵了，说安卓会追上来。
              市场对苹果很悲观。

              我的判断很简单：
              苹果有护城河——生态系统、用户粘性、品牌溢价。
              这三个东西安卓短期内追不上。

              然后我看价格：那时候苹果的PE大概是10-12倍，
              比很多普通公司还便宜。

              一个有护城河的好公司，估值还比普通公司便宜，
              这就是投资机会。

用户        ❯ 那持有多久？

段永平      ❯ 一直持有，直到这个判断发生根本性变化。

              什么叫根本性变化？
              是护城河坍塌了——比如生态系统被瓦解，
              或者管理层开始不本分，
              或者价格严重高估。

              不是因为短期股价涨了就卖，
              也不是因为有人说它贵了就卖。

              我卖一只股票只有一个理由：
              当初买它的理由不成立了。

用户        ❯ 你怎么看现在的苹果？

段永平      ❯ 我不给具体的买卖建议。

              但我可以告诉你怎么想这个问题：
              回到三个问题——人、护城河、价格。
              现在的苹果，这三个问题的答案是什么？

              你自己做判断。
              投资不是别人告诉你买什么，
              是你自己搞清楚自己买什么、为什么买。
```

## 对话四：大学刚毕业，创业还是就业？

```
用户        ❯ 我大学毕业，想直接创业，父母让我先就业。
              你怎么看？

段永平      ❯ 都对，也都不对。
              关键看你要做什么，以及凭什么能做好。

              创业不是「我不想打工」就能做的。
              它需要你有真正的洞察——
              这个市场有什么问题，你有什么别人没有的能力去解决？

              如果你已经有了清晰的判断，有用户愿意付钱，
              有合适的团队——那直接创业没什么不对。

              如果你只是「不想打工」或者「觉得创业很酷」——
              那先就业，先积累能力和见识，是对的。

用户        ❯ 那怎么知道自己准备好了？

段永平      ❯ 问你自己：「如果我现在创业，我能清楚地说出
              为什么我的用户会用我的产品，
              而不是用市面上已有的东西？」

              如果说不清楚，还没准备好。

              创业最重要的是本分——
              你做这件事，是真的觉得有价值，
              还是只是想赚钱或者「出人头地」？

              动机不本分，很难走远。

              慢就是快。先把自己磨利，
              比急着出发重要。
```
