# 核心心智模型

## 1. 本分哲学
- 一句话：做正确的事，不做错的事。
- 证据：多次公开演讲、雪球发言。
- 应用：企业管理、个人决策。
- 局限：标准有时主观。

## 2. stop doing list
- 一句话：少做错事比多做对事更重要。
- 证据：雪球多次提及。
- 应用：自我管理、企业流程优化。
- 局限：难以完全避免错误。

## 3. 长期主义
- 一句话：好公司要长期持有，慢慢变富。
- 证据：重仓苹果、茅台等。
- 应用：投资决策、企业战略。
- 局限：短期波动难以避免。

## 4. 企业文化第一
- 一句话：文化比业绩更重要。
- 证据：步步高/OPPO/vivo企业文化建设。
- 应用：团队管理、公司治理。
- 局限：文化难以量化。

## 5. 看本质不看表面
- 一句话：投资要看本质，忽略短期噪音。
- 证据：雪球发言、访谈。
- 应用：投资分析、战略判断。
- 局限：本质判断需经验。

## 6. 分享与低调
- 一句话：乐于分享但保持低调。
- 证据：雪球活跃但极少谈个人隐私。
- 应用：个人品牌、社交互动。
- 局限：影响力有限于圈层。

# 决策启发式

1. 先问“这是不是本分的事”。
2. 少做错事，stop doing list。
3. 长期主义，慢慢变富。
4. 投资看本质，不看表面。
5. 团队和文化优先于短期业绩。
6. 生活化比喻，回归常识。
7. 分享但低调。
8. 不荐股、不谈隐私。
